# Go 结构体与方法 (Struct & Methods)

结构体是 Go 语言中最重要的复合数据类型之一，它是实现面向对象编程的基础。

---

## 1. 结构体基础

### 1.1 什么是结构体？

结构体是将多个不同类型的数据组合在一起的自定义类型。类似于其他语言中的"类"，但更简单。

```go
// 定义一个表示"人"的结构体
type Person struct {
    Name   string
    Age    int
    Email  string
    Active bool
}
```

### 1.2 创建结构体实例

Go 提供了多种创建结构体实例的方式：

```go
// 方式1：使用字段名（推荐）
p1 := Person{
    Name:   "Alice",
    Age:    25,
    Email:  "alice@example.com",
    Active: true,
}

// 方式2：按顺序赋值（不推荐，容易出错）
p2 := Person{"Bob", 30, "bob@example.com", false}

// 方式3：先声明，后赋值
var p3 Person
p3.Name = "Charlie"
p3.Age = 35

// 方式4：使用 new() 返回指针
p4 := new(Person)  // p4 是 *Person 类型
p4.Name = "David"

// 方式5：取地址
p5 := &Person{Name: "Eve", Age: 28}  // p5 是 *Person 类型
```

### 1.3 零值

未初始化的结构体字段会被设为零值：
- 数值类型 → `0`
- 字符串 → `""`（空字符串）
- 布尔类型 → `false`
- 指针/切片/map → `nil`

```go
var p Person
fmt.Println(p.Name)   // ""
fmt.Println(p.Age)    // 0
fmt.Println(p.Active) // false
```

## 2. 访问和修改字段

### 2.1 点操作符

使用 `.` 访问结构体字段：

```go
person := Person{Name: "Alice", Age: 25}

// 读取
fmt.Println(person.Name)  // Alice

// 修改
person.Age = 26
fmt.Println(person.Age)   // 26
```

### 2.2 指针访问

Go 会自动解引用指针，语法一致：

```go
p := &Person{Name: "Bob", Age: 30}

// 这两种写法等价
fmt.Println(p.Name)     // 自动解引用
fmt.Println((*p).Name)  // 显式解引用
```

## 3. 结构体与函数

### 3.1 结构体作为参数

**值传递**（传递副本）：

```go
func birthday(p Person) {
    p.Age++  // 只修改副本，原结构体不变
}

alice := Person{Name: "Alice", Age: 25}
birthday(alice)
fmt.Println(alice.Age)  // 仍然是 25
```

**指针传递**（传递引用）：

```go
func birthday(p *Person) {
    p.Age++  // 修改原结构体
}

alice := Person{Name: "Alice", Age: 25}
birthday(&alice)
fmt.Println(alice.Age)  // 变成 26
```

### 3.2 何时使用指针？

| 场景 | 建议 |
|------|------|
| 需要修改原结构体 | 使用指针 |
| 结构体很大（避免复制开销） | 使用指针 |
| 只读取小结构体 | 可以使用值 |
| 保持一致性 | 建议统一使用指针 |

## 4. 方法 (Methods)

### 4.1 什么是方法？

方法是绑定到特定类型的函数。与函数的区别在于方法有一个**接收者（receiver）**。

```go
// 函数
func Greet(p Person) string {
    return "Hello, " + p.Name
}

// 方法
func (p Person) Greet() string {
    return "Hello, " + p.Name
}

// 调用方式不同
person := Person{Name: "Alice"}
Greet(person)     // 函数调用
person.Greet()    // 方法调用
```

### 4.2 值接收者 vs 指针接收者

**值接收者**：操作副本，不影响原结构体

```go
func (p Person) GetAge() int {
    return p.Age
}

func (p Person) SetAge(age int) {
    p.Age = age  // ⚠️ 修改的是副本！原结构体不变
}
```

**指针接收者**：操作原结构体

```go
func (p *Person) SetAge(age int) {
    p.Age = age  // ✅ 修改原结构体
}

func (p *Person) Birthday() {
    p.Age++
}
```

### 4.3 接收者选择指南

```go
// ✅ 使用指针接收者的情况：
// 1. 需要修改结构体
func (p *Person) SetName(name string) {
    p.Name = name
}

// 2. 结构体很大，避免复制
type BigData struct {
    data [1000000]int
}
func (b *BigData) Process() { /* ... */ }

// 3. 保持一致性（如果一个方法用指针，其他也用指针）
```

### 4.4 方法可以定义在任何类型上

```go
// 为基本类型创建别名
type MyInt int

func (m MyInt) Double() MyInt {
    return m * 2
}

func (m *MyInt) Triple() {
    *m = *m * 3
}

num := MyInt(5)
fmt.Println(num.Double())  // 10
num.Triple()
fmt.Println(num)           // 15
```

## 5. 结构体嵌入（组合）

Go 没有继承，但通过**嵌入**实现组合，达到类似效果。

### 5.1 匿名嵌入

```go
type Address struct {
    City    string
    Country string
}

type Person struct {
    Name    string
    Age     int
    Address // 匿名嵌入
}

// 使用
p := Person{
    Name: "Alice",
    Age:  25,
    Address: Address{
        City:    "Beijing",
        Country: "China",
    },
}

// 可以直接访问嵌入字段的属性
fmt.Println(p.City)     // 等同于 p.Address.City
fmt.Println(p.Country)  // 等同于 p.Address.Country
```

### 5.2 方法提升

嵌入结构体的方法会自动提升到外层：

```go
func (a Address) FullAddress() string {
    return a.City + ", " + a.Country
}

// Person 自动获得 Address 的方法
p := Person{
    Name:    "Alice",
    Address: Address{City: "Beijing", Country: "China"},
}

fmt.Println(p.FullAddress())  // 直接调用！
```

### 5.3 命名嵌入 vs 匿名嵌入

```go
// 命名嵌入
type Employee struct {
    Name    string
    Address Address  // 命名字段
}

emp := Employee{Name: "Bob", Address: Address{City: "Shanghai"}}
fmt.Println(emp.Address.City)  // 必须通过 Address 访问

// 匿名嵌入
type Manager struct {
    Name string
    Address  // 匿名嵌入
}

mgr := Manager{Name: "Charlie", Address: Address{City: "Guangzhou"}}
fmt.Println(mgr.City)  // 可以直接访问
```

## 6. 结构体标签 (Tags)

结构体标签用于在运行时通过反射读取元信息，常用于序列化。

### 6.1 JSON 标签

```go
type User struct {
    ID        int    `json:"id"`
    Username  string `json:"username"`
    Password  string `json:"-"`                    // 忽略此字段
    Email     string `json:"email,omitempty"`      // 空值时忽略
    CreatedAt string `json:"created_at,omitempty"`
}

user := User{
    ID:       1,
    Username: "alice",
    Password: "secret123",
    Email:    "alice@example.com",
}

// 编码为 JSON
data, _ := json.Marshal(user)
fmt.Println(string(data))
// 输出: {"id":1,"username":"alice","email":"alice@example.com"}
// 注意：Password 被忽略了
```

### 6.2 常用标签

| 标签 | 用途 |
|------|------|
| `json:"name"` | JSON 字段名 |
| `json:"-"` | 忽略字段 |
| `json:",omitempty"` | 空值时省略 |
| `db:"column_name"` | 数据库列名 |
| `validate:"required"` | 字段验证 |
| `form:"field_name"` | 表单字段名 |

## 7. 结构体比较

### 7.1 可比较的结构体

如果结构体所有字段都是可比较的，则结构体可以用 `==` 比较：

```go
type Point struct {
    X, Y int
}

p1 := Point{1, 2}
p2 := Point{1, 2}
p3 := Point{3, 4}

fmt.Println(p1 == p2)  // true
fmt.Println(p1 == p3)  // false
```

### 7.2 不可比较的结构体

包含切片、map、函数的结构体不能直接比较：

```go
type Data struct {
    Values []int  // 切片不可比较
}

d1 := Data{Values: []int{1, 2, 3}}
d2 := Data{Values: []int{1, 2, 3}}
// d1 == d2  // ❌ 编译错误！

// 需要使用 reflect.DeepEqual
import "reflect"
fmt.Println(reflect.DeepEqual(d1, d2))  // true
```

## 9. 常见错误

### 9.1 忘记使用指针接收者

```go
// ❌ 错误
func (p Person) SetName(name string) {
    p.Name = name  // 修改的是副本
}

// ✅ 正确
func (p *Person) SetName(name string) {
    p.Name = name
}
```

### 9.2 循环中取地址问题

```go
persons := []Person{{Name: "A"}, {Name: "B"}, {Name: "C"}}
var pointers []*Person

// ❌ 错误：所有指针都指向同一个地址
for _, p := range persons {
    pointers = append(pointers, &p)
}

// ✅ 正确方式 1：使用索引
for i := range persons {
    pointers = append(pointers, &persons[i])
}

// ✅ 正确方式 2：创建副本
for _, p := range persons {
    p := p  // 创建新变量
    pointers = append(pointers, &p)
}
```

### 9.3 nil 结构体指针调用方法

```go
var p *Person  // nil 指针

// 值接收者方法会 panic
// p.GetName()  // panic: nil pointer dereference

// 但可以定义能处理 nil 的方法
func (p *Person) SafeName() string {
    if p == nil {
        return "Unknown"
    }
    return p.Name
}
```

## 10. 小结

| 概念 | 要点 |
|------|------|
| 结构体 | 组合不同类型数据的自定义类型 |
| 方法 | 绑定到类型的函数，有接收者 |
| 值接收者 | 操作副本，不修改原结构体 |
| 指针接收者 | 操作原结构体，可以修改 |
| 嵌入 | Go 的组合方式，替代继承 |
| 标签 | 元信息，用于序列化等 |
