## 1. 环境搭建

### 1.1 安装 Go

#### macOS (推荐使用 Homebrew)

```bash
# 安装
brew install go

# 验证安装
go version
# 输出类似: go version go1.22.0 darwin/arm64
```

#### 其他安装方式

- **官网下载**: https://go.dev/dl/
- **Linux**: `sudo apt install golang` 或下载二进制包
- **Windows**: 下载 MSI 安装包

### 1.2 配置环境变量

Go 1.11+ 引入了 Go Modules，大部分情况下不需要配置 `GOPATH`，但了解这些概念仍然有用：

```bash
# 查看 Go 环境配置
go env

# 重要的环境变量
go env GOROOT   # Go 安装目录
go env GOPATH   # 工作目录 (默认 ~/go)
go env GOPROXY  # 模块代理
```

#### 配置国内代理 (加速依赖下载)

```bash
# 设置代理
go env -w GOPROXY=https://goproxy.cn,direct

# 或在 ~/.zshrc 中添加
export GOPROXY=https://goproxy.cn,direct
```

### 1.3 编辑器配置

#### Cursor / VS Code

1. 安装 Go 扩展
2. 打开命令面板 (Cmd+Shift+P)，运行 `Go: Install/Update Tools`
3. 选择全部工具并安装

推荐安装的工具：
- `gopls` - 语言服务器
- `dlv` - 调试器
- `staticcheck` - 静态分析

### 1.4 创建第一个项目

```bash
# 创建项目目录
mkdir hello-go && cd hello-go

# 初始化 Go 模块
go mod init hello-go

# 创建主文件
touch main.go
```

## 2. Hello World 详解

### 2.1 基本程序结构

```go
// main.go
package main        // 声明包名，main 包是程序入口

import "fmt"        // 导入标准库的 fmt 包

func main() {       // main 函数是程序执行的起点
    fmt.Println("Hello, Go!")
}
```

### 2.2 逐行解析

| 代码 | 说明 |
|------|------|
| `package main` | 每个 Go 文件必须声明所属包。`main` 包表示可执行程序 |
| `import "fmt"` | 导入格式化 I/O 包，用于打印输出 |
| `func main()` | 程序入口点，无参数无返回值 |
| `fmt.Println()` | 打印并换行 |

### 2.3 运行程序

```bash
# 方式1: 直接运行 (编译+执行)
go run main.go

# 方式2: 先编译，再执行
go build -o hello main.go
./hello

# 方式3: 安装到 $GOPATH/bin
go install
```

### 2.4 多文件导入

```go
import (
    "fmt"
    "os"
    "strings"
)
```

### 2.5 包的可见性规则

Go 使用**首字母大小写**控制可见性：

- **大写字母开头** → 公开 (可被外部包访问)
- **小写字母开头** → 私有 (仅包内可见)

```go
package mypackage

var PublicVar = "可被外部访问"    // 公开
var privateVar = "仅包内可见"     // 私有

func PublicFunc() {}              // 公开
func privateFunc() {}             // 私有
```

## 3. 变量与常量

### 3.1 变量声明方式

Go 提供多种变量声明方式：

```go
package main

import "fmt"

func main() {
    // 方式1: 完整声明
    var name string = "Go"
    
    // 方式2: 类型推断
    var age = 15
    
    // 方式3: 短声明 (最常用，仅限函数内)
    language := "Golang"
    
    // 方式4: 声明多个变量
    var x, y int = 10, 20
    
    // 方式5: 批量声明
    var (
        city    = "Beijing"
        country = "China"
        code    = 86
    )
    
    fmt.Println(name, age, language)
    fmt.Println(x, y)
    fmt.Println(city, country, code)
}
```

### 3.2 变量的零值

Go 中未初始化的变量会被赋予**零值**：

| 类型 | 零值 |
|------|------|
| `int`, `float64` 等数值类型 | `0` |
| `bool` | `false` |
| `string` | `""` (空字符串) |
| 指针、切片、map、channel、函数、接口 | `nil` |

```go
var i int       // 0
var f float64   // 0
var b bool      // false
var s string    // ""

fmt.Printf("int: %d, float: %f, bool: %t, string: %q\n", i, f, b, s)
```

### 3.3 常量

常量使用 `const` 声明，必须在编译时确定值：

```go
// 单个常量
const Pi = 3.14159
const Greeting = "Hello"

// 批量声明
const (
    StatusOK       = 200
    StatusNotFound = 404
    StatusError    = 500
)

// iota: 常量生成器
const (
    Sunday    = iota  // 0
    Monday           // 1
    Tuesday          // 2
    Wednesday        // 3
    Thursday         // 4
    Friday           // 5
    Saturday         // 6
)

// iota 高级用法
const (
    _  = iota             // 忽略 0
    KB = 1 << (10 * iota) // 1 << 10 = 1024
    MB                    // 1 << 20 = 1048576
    GB                    // 1 << 30
    TB                    // 1 << 40
)
```

### 3.4 变量作用域

```go
package main

var globalVar = "全局变量"  // 包级变量

func main() {
    localVar := "局部变量"
    
    if true {
        blockVar := "块级变量"
        fmt.Println(blockVar)  // OK
    }
    // fmt.Println(blockVar)  // 错误: blockVar 不可见
    
    fmt.Println(globalVar, localVar)
}
```

## 4. 基本数据类型

### 4.1 数值类型

#### 整数类型

| 类型 | 大小 | 范围 |
|------|------|------|
| `int8` | 8位 | -128 ~ 127 |
| `int16` | 16位 | -32768 ~ 32767 |
| `int32` | 32位 | -2^31 ~ 2^31-1 |
| `int64` | 64位 | -2^63 ~ 2^63-1 |
| `int` | 32/64位 | 取决于系统 |
| `uint8` | 8位 | 0 ~ 255 |
| `uint16` | 16位 | 0 ~ 65535 |
| `uint32` | 32位 | 0 ~ 2^32-1 |
| `uint64` | 64位 | 0 ~ 2^64-1 |
| `uint` | 32/64位 | 取决于系统 |

```go
var i int = 42
var i8 int8 = 127
var u uint = 100
var u32 uint32 = 4294967295

// 类型转换 (必须显式)
var a int32 = 10
var b int64 = int64(a)
```

#### 浮点数类型

| 类型 | 大小 | 精度 |
|------|------|------|
| `float32` | 32位 | 约 7 位有效数字 |
| `float64` | 64位 | 约 15 位有效数字 |

```go
var f1 float32 = 3.14
var f2 float64 = 3.141592653589793

// 默认推断为 float64
pi := 3.14159
```

### 4.2 布尔类型

```go
var isTrue bool = true
var isFalse bool = false

// 布尔运算
result := true && false  // false
result = true || false   // true
result = !true           // false
```

### 4.3 字符串类型

```go
// 字符串是不可变的
s := "Hello, 世界"

// 获取长度 (字节数)
fmt.Println(len(s))  // 13 (英文1字节，中文3字节)

// 获取字符数
fmt.Println(utf8.RuneCountInString(s))  // 9

// 字符串拼接
s1 := "Hello"
s2 := "World"
s3 := s1 + ", " + s2

// 多行字符串
multiLine := `这是
多行
字符串`

// 字符串索引 (返回字节)
fmt.Println(s[0])  // 72 (H 的 ASCII)

// 遍历字符串
for i, r := range s {
    fmt.Printf("索引: %d, 字符: %c, 码点: %d\n", i, r, r)
}
```

### 4.4 byte 和 rune

```go
// byte 是 uint8 的别名，表示一个字节
var b byte = 'A'  // 65

// rune 是 int32 的别名，表示一个 Unicode 码点
var r rune = '中'  // 20013

// 字符串与 []byte 转换
s := "hello"
bytes := []byte(s)
s2 := string(bytes)

// 字符串与 []rune 转换 (处理中文)
s = "你好"
runes := []rune(s)
fmt.Println(len(runes))  // 2
```

### 4.5 类型转换

Go 不支持隐式类型转换，必须显式转换：

```go
var i int = 42
var f float64 = float64(i)
var u uint = uint(f)

// 字符串与数字转换需要用 strconv 包
import "strconv"

// int → string
s := strconv.Itoa(42)  // "42"

// string → int
i, err := strconv.Atoi("42")
if err != nil {
    fmt.Println("转换失败")
}

// float → string
s = strconv.FormatFloat(3.14, 'f', 2, 64)  // "3.14"

// string → float
f, err = strconv.ParseFloat("3.14", 64)
```

## 5. 运算符

### 5.1 算术运算符

```go
a, b := 10, 3

fmt.Println(a + b)   // 13 加法
fmt.Println(a - b)   // 7  减法
fmt.Println(a * b)   // 30 乘法
fmt.Println(a / b)   // 3  除法 (整数除法)
fmt.Println(a % b)   // 1  取模

// 自增/自减 (只能作为语句，不能用于表达式)
a++  // a = 11
b--  // b = 2

// 错误示例:
// x := a++  // 编译错误
```

### 5.2 比较运算符

```go
a, b := 10, 20

fmt.Println(a == b)  // false 等于
fmt.Println(a != b)  // true  不等于
fmt.Println(a < b)   // true  小于
fmt.Println(a > b)   // false 大于
fmt.Println(a <= b)  // true  小于等于
fmt.Println(a >= b)  // false 大于等于
```

### 5.3 逻辑运算符

```go
x, y := true, false

fmt.Println(x && y)  // false 与
fmt.Println(x || y)  // true  或
fmt.Println(!x)      // false 非
```

### 5.4 位运算符

```go
a, b := 5, 3  // 二进制: 101, 011

fmt.Println(a & b)   // 1  按位与 (001)
fmt.Println(a | b)   // 7  按位或 (111)
fmt.Println(a ^ b)   // 6  按位异或 (110)
fmt.Println(a << 1)  // 10 左移 (1010)
fmt.Println(a >> 1)  // 2  右移 (10)
```

### 5.5 赋值运算符

```go
a := 10

a += 5   // a = a + 5   → 15
a -= 3   // a = a - 3   → 12
a *= 2   // a = a * 2   → 24
a /= 4   // a = a / 4   → 6
a %= 4   // a = a % 4   → 2
a &= 3   // a = a & 3
a |= 4   // a = a | 4
a ^= 2   // a = a ^ 2
a <<= 1  // a = a << 1
a >>= 1  // a = a >> 1
```

## 6. 控制流

### 6.1 if-else

```go
age := 18

// 基本形式
if age >= 18 {
    fmt.Println("成年人")
} else {
    fmt.Println("未成年人")
}

// 多分支
score := 85
if score >= 90 {
    fmt.Println("优秀")
} else if score >= 80 {
    fmt.Println("良好")
} else if score >= 60 {
    fmt.Println("及格")
} else {
    fmt.Println("不及格")
}

// 带初始化语句 (变量作用域限于 if 块)
if num := calculate(); num > 0 {
    fmt.Println("正数:", num)
} else {
    fmt.Println("非正数:", num)
}
// num 在这里不可见
```

### 6.2 for 循环

Go 只有 `for` 一种循环，但能实现多种形式：

```go
// 形式1: 经典 for
for i := 0; i < 5; i++ {
    fmt.Println(i)
}

// 形式2: while 风格
count := 0
for count < 5 {
    fmt.Println(count)
    count++
}

// 形式3: 无限循环
for {
    fmt.Println("无限循环")
    break  // 使用 break 跳出
}

// 形式4: range 遍历
nums := []int{1, 2, 3, 4, 5}
for index, value := range nums {
    fmt.Printf("索引: %d, 值: %d\n", index, value)
}

// 忽略索引
for _, value := range nums {
    fmt.Println(value)
}

// 忽略值
for index := range nums {
    fmt.Println(index)
}

// 遍历字符串
for i, char := range "Hello, 世界" {
    fmt.Printf("%d: %c\n", i, char)
}

// 遍历 map
m := map[string]int{"a": 1, "b": 2}
for key, value := range m {
    fmt.Printf("%s: %d\n", key, value)
}
```

### 6.3 break、continue、goto

```go
// break: 跳出循环
for i := 0; i < 10; i++ {
    if i == 5 {
        break
    }
    fmt.Println(i)  // 0, 1, 2, 3, 4
}

// continue: 跳过本次迭代
for i := 0; i < 5; i++ {
    if i == 2 {
        continue
    }
    fmt.Println(i)  // 0, 1, 3, 4
}

// 带标签的 break (跳出外层循环)
outer:
for i := 0; i < 3; i++ {
    for j := 0; j < 3; j++ {
        if i == 1 && j == 1 {
            break outer
        }
        fmt.Printf("i=%d, j=%d\n", i, j)
    }
}

// goto (谨慎使用)
func example() {
    i := 0
start:
    if i < 5 {
        fmt.Println(i)
        i++
        goto start
    }
}
```

### 6.4 switch

```go
// 基本 switch
day := "Monday"
switch day {
case "Monday":
    fmt.Println("周一")
case "Tuesday":
    fmt.Println("周二")
case "Saturday", "Sunday":  // 多个值
    fmt.Println("周末")
default:
    fmt.Println("工作日")
}

// switch 不需要 break，自动终止
// 如需穿透，使用 fallthrough
switch num := 2; num {
case 1:
    fmt.Println("一")
    fallthrough
case 2:
    fmt.Println("二")
    fallthrough
case 3:
    fmt.Println("三")
}
// 输出: 二 三

// 无条件 switch (替代 if-else 链)
score := 85
switch {
case score >= 90:
    fmt.Println("A")
case score >= 80:
    fmt.Println("B")
case score >= 70:
    fmt.Println("C")
default:
    fmt.Println("D")
}

// 类型 switch (后续学习接口时会用到)
var x interface{} = "hello"
switch v := x.(type) {
case int:
    fmt.Println("整数:", v)
case string:
    fmt.Println("字符串:", v)
default:
    fmt.Println("未知类型")
}
```

## 7. 函数基础

### 7.1 函数定义

```go
// 基本语法
func 函数名(参数列表) 返回值类型 {
    // 函数体
}

// 无参数无返回值
func sayHello() {
    fmt.Println("Hello!")
}

// 有参数有返回值
func add(a int, b int) int {
    return a + b
}

// 参数类型相同可简写
func multiply(a, b int) int {
    return a * b
}

// 多返回值
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("除数不能为零")
    }
    return a / b, nil
}

// 使用多返回值
result, err := divide(10, 2)
if err != nil {
    fmt.Println("错误:", err)
} else {
    fmt.Println("结果:", result)
}
```

### 7.2 命名返回值

```go
func rectangle(width, height int) (area, perimeter int) {
    area = width * height
    perimeter = 2 * (width + height)
    return  // 裸返回，返回命名的返回值
}

a, p := rectangle(5, 3)
fmt.Printf("面积: %d, 周长: %d\n", a, p)
```

### 7.3 可变参数

```go
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

// 调用
fmt.Println(sum(1, 2, 3))       // 6
fmt.Println(sum(1, 2, 3, 4, 5)) // 15

// 传入切片
numbers := []int{1, 2, 3, 4}
fmt.Println(sum(numbers...))    // 使用 ... 展开切片
```

### 7.4 匿名函数

```go
// 定义并立即执行
func() {
    fmt.Println("匿名函数")
}()

// 赋值给变量
greet := func(name string) {
    fmt.Printf("Hello, %s!\n", name)
}
greet("Go")

// 作为参数传递
func apply(fn func(int, int) int, a, b int) int {
    return fn(a, b)
}

result := apply(func(x, y int) int {
    return x + y
}, 3, 4)
fmt.Println(result)  // 7
```

### 7.5 闭包

闭包是引用了外部变量的函数：

```go
func counter() func() int {
    count := 0
    return func() int {
        count++
        return count
    }
}

c := counter()
fmt.Println(c())  // 1
fmt.Println(c())  // 2
fmt.Println(c())  // 3

c2 := counter()
fmt.Println(c2()) // 1 (新的计数器)
```

### 7.6 defer

`defer` 延迟执行，常用于资源清理：

```go
func example() {
    defer fmt.Println("1. 最后执行")
    defer fmt.Println("2. 倒数第二")
    defer fmt.Println("3. 倒数第三")
    fmt.Println("4. 首先执行")
}
// 输出顺序: 4, 3, 2, 1 (LIFO 后进先出)

// 常见用途: 关闭文件
func readFile(filename string) {
    file, err := os.Open(filename)
    if err != nil {
        return
    }
    defer file.Close()  // 确保函数结束时关闭文件
    
    // 读取文件...
}
```

    for i := 2; i <= n; i++ {
        a, b = b, a+b
    }
    return b
}
